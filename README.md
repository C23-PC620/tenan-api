# tenan-api
How to use :
1. move to tenan-api folder
```
$ cd tenan-api
```
2. run NPM install
```
$ npm install
```
3. Edit the environment variable in .env file
4. Create database and tables from database/database.sql 
5. run NPM run start-dev
```
$ npm run start-dev
```
